Operatörler:

_* : array içindeki tüm değerler
<- : to anlamında 
:: append anlamında  (tamamen bunlarla array tanımlanacaksa sonunda :: Nil olmalı)
::: iki list'i append etmek için kullanılır
=> : hof'larda to gibi kullanılır // match case içn ise anlamında casedeki iafededen sonra kullanılır
-> : maplemede key value gösterimi için kullanılır
map(_._1) : tüm keyleri getiri
map(_._2) : tüm valueları getirir

- Bir class'ın constructor'ını kullanmak istersen "new" kullanmalısın, case class yazarsan gerekmek
- "this" hem auxiliary constructor yazmada hem de mevcut class'ın objesini çağırmada kullanılır
- higher order functionlar pythondaki lambda fonksiyonlar gibi: val myfunc: Int => Int = a => a * a

- Alttaki HOF'Lar aynı:

l.reduce((l,r) => l+r)
l.foldLeft(0)((l,r) => l+r)


- prepended(0): append gibi ama sonuna değil başına 0 ekliyor
